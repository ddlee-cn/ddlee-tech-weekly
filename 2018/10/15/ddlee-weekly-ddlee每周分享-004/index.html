2018-10-15

国庆期间度假去了，鸽了两期，自责一番。另外开辟了一个“东东月报”，关于自己的学习和阅读总结，可在这里订阅和查看样刊。

Blog Post/博文

What’s a senior engineer’s job? - Julia Evans

讨论了一个高级工程师的工作内容，以及和管理者工作内容的区别。为自己划定工作内容的边界是重要的。

Going sponsor-free | 3blue1brown on Patreon

著名的数学科普类YouTube频道3blue1brown宣布不再接受广告赞助，直接原因是通过Patreon平台，频道观众的资助已经能够维持频道的运营了。这是一件非常好的事情。文章里频道主简要回顾了YouTuber们的经济收入来源演进的历程，也指出广告赞助的形式会影响观众的注意力，直接接受来自观众而非广告商的资助是符合Incentive alignment的。这种捐助者资助、内容却分享给所有人的运营模式的确在相当程度上达到了各方都受益的效果，但也赖于一个对知识足够尊重的文化环境。

Open Source/开源项目

cms-flash/beauty-net

基于PyTorch，提供了网络、优化器、评测指标等类和训练、推断脚本，可以作为项目的Starter Code。

Apps/应用推荐

All Time • All categories • Top Medium Stories • TrendBar

Medium文章的排行榜，可以按类别、时间等查看。

oasisfeng/island

一直在用的沙盒app island开源了。它可以分离一部分应用到另一个用户下（称为work mode），有效地冻结应用。

viatsko/awesome-vscode

VS Code的插件推荐列表。个人目前是用它作为主力文本编辑器的，一是相比atom的启动速度，二是相比sublime text的开源和插件系统。

Podflix: Search and discovery engine for podcasts

一个第三方的Podcast集合站，可以方便地搜索到Podcast订阅源，不必受所用客户端的局限。

Tutorials/教程

How to work out the missing dependencies for a Python package - piwheels blog

很实用的一个技巧：通过跟踪Python库的连接位置来找到缺失dependency的原因，进而通过apt-file来安装dependency。

News/快讯

Windows 10 October 2018 Update: the 10 best new features

Win10 10月更新的特性总结，比较有吸引力的是电源管理。但由于一系列BUG最近暂停了推送，仍在观望中。
