2019-04-01

写在前面：

很抱歉，因为考试周、过年、换新环境以及忙，每周分享拖更了长达两个月的时间。但我依旧还是想坚持这个事情，所以这不是一份停更的通知，而是新的一期，尽管里面的文章很多并不够新了。

我还是想再谈一下我为什么要维护这样一个博客，在WWW式微、注意力被建起围墙的APP分割的时代。

我想来想去，从中文技术内容的生态、到大众阅读习惯、再到内容的编辑与分发，但浮现在我脑海中最核心的一个理由是，我相信好内容值得更多的人读。

但，好内容也应该被保护。而不是被抄袭、篡改，博得流量，为广告做嫁衣。因此我选择了这样的形式——分享原文链接+评论，并通过邮件订阅和RSS订阅两种极为开放的方式分发。我写的评论其实不好，但若能帮助别人注意到某些高质量的内容，那便是十分美好了。

鉴于我的工作周期和时间，仍然会在周一周二这两天择时更新。感谢您的关注。

Blog Post/博文

Microsoft, Facebook, trust and privacy
很有趣的视角：Facebook上充斥的假新闻，跟Windows上曾经流行的病毒和木马程序是一样的模式，而两家公司的处理方式也是类似的。

Redesigning Github repository page
对GitHub用户页面的重新设计，条分缕析，虽然每一步都是一小点，但对比前后变化确实清爽高效很多，可以管窥设计师思维。

The MIT License, Line by Line
对MIT许可协议的深度解读，开源协议的设计确实是很精妙的工作。

Open Source/开源项目

TuSimple/simpledet
TuSimple开源的基于MXNet的检测框架，整体上借鉴Detectron的痕迹非常明显（Detectron基于Facebook的Caffe2，后者现已并入PyTorch）。不管怎么说，还是开心和感激的。

Apps/应用推荐

wowamusic
不必为版权问题发愁（Royal-Free）的音乐收集站点。

My Brand New Logo — My Brand New Logo – logo maker | create your own logo
在线生成个性logo，可以个性化调整并能多格式导出，对英文的支持不错。logo一般是要以矢量图存在，想必需要很多hack才能做得如此可用的产品。

Tutorials/教程

Managing Multiple Python Versions With pyenv – Real Python
pyenv可以作为pipenv的补充，管理全局的python版本。

Supercharge Your Classes With Python super() – Real Python
讲解了Python类定义中super()的用法，收藏备查。

Troubleshooting Deep Neural Networks
一份深度网络模型的调试指南。建议的使用方法是通读后把握调试环节的大体思路，遇到问题时再参考其列出的常见问题清单（一次性细读容易过载反而得到很少）。

Hacker Tools
“黑客”一词在中文环境里被相当地污名化了，与“鼓捣电脑的”构成大众对IT学生/从业者的刻板印象之两端：前者施加道德之咒，后者则是市侩心理的投射。在堪当黑客精神源泉之一的MIT开设的这一课程中，老师们整理了Hacker常用的工具，主题包括命令行工具、文本编辑器、远程主机管理等，这十几个小时的精力投入实在非常值得。顺便，推荐《黑客与画家》来了解Hacker文化。
